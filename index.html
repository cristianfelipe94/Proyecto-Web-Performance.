WebPerformance.

¿Qué es?

Es la disciplina que se encarga de analizar un sitio web para recomendar cambios que ayuden a optimizar la velocidad de carga de la misma. La velocidad es un elemento crítico para ejecutar un sitio web exitoso y siempre debe ser una prioridad para los administradores de sitios web. Los sitios web de carga más rápida se benefician de un mejor compromiso del usuario. La velocidad de carga de una Web es un factor a tener en cuenta para el posicionamiento en buscadores.

¿Quienes somos?

Estudiantes de Diseño y Desarrollo Web del CETAV, por medio de este proyecto queremos dar a conocer las pricipales practicas que se llevan a cabo para lograr un buen web Performance para mejorar la experiencia de usuario, esto mediante elaboracion de un micrositio informativo.

Compresión de archivos.

A medida que las páginas web crecen en complejidad, también lo hacen sus archivos y posteriormente, sus tiempos de carga. Por eso hay que tener en cuenta optimizar imagenes, videos, audio y comprimir archivos PDF de ser necesario, para tener una buena experiencia en el sitio web.

Herramientas recomendadas a utilizar:

Compressor.io: Es un optimizador de imagenes basado en servicios de la nube.

Convert.io: Es una herramienta gratis que nos permite comprimir archivos PDF en tres calidades distintas.

Online Audio Converter: Es un sencillo online, puede convertir archivos de formatos pesados a otros no tanto, como .wav en .mp3, se puede reducir la calidad de un mp3 con pérdida en resolución casi imperceptible.

Handbrake: Es un programa gratuito que permite cambiar el formato de los videos sin limites de tamaño, ya que otros servicios o softwares nos da un límite de tamaño al subir el archivo.

Reducción del código.

Hoy en día la web moderna requiere y exige de archivos de poco tamaño, para que los dispositivos móviles puedan leer las páginas a mayor velocidad, consumir menor ancho de banda y disminuir en lo posible el consumo de datos. Existen herramientas y buenas prácticas que nos pueden ayudar a reducir el código.

Links de herramientas las cuales nos dejan comprimir y descomprimir el código HTML, Javascript y CSS.

Comprimir codigo HTML. Comprimir codigo Javascript. Comprimir codigo CSS.

Optimización de la imagen.

Como se nombra anteriormente, optimizar y comprimir los archivos es un impacto importante en la carga de las paginas web, por eso no cargue las fotos originales en su sitio web, son demasiado pesadas ya que contienen mucha información adicional (como metadatos acerca de la cámara, fecha, ubicación...) que pueden ser eliminados sin que eso suponga una pérdida de calidad. Use herramientas que reduzcan el tamaño de la imagen mientras que la calidad permanece más o menos igual.

Herramienta recomendada a utilizar:

Riot (Radical Image Optimization Tool): Es una herramienta para optimizar las imágenes desde tu ordenador antes de subirlas. Es gratis para Windows, pero si usa MAC, se puede usarImageOptim.

Errores 404.

Cualquier archivo faltante genera un error de HTTP 404. El valor 404 advierte de que la URL solicitada no se encuentra en el servidor, como cuando intentas acceder a una página que ha sido borrada o escribes mal la dirección en el navegador.

Los errores 404, pueden ser bastante agotadores para el servidor. Existen varias aplicaciones recomendadas para detectar este tipo de errores, cuya función principal es localizar los enlaces que estan fallando en tu página o enlaces rotos.
